== Description ==

Improve your Business with a WhatsApp Chat Button. It allows your Customer to contact you with whatsapp.


= Features =
More options will be added soon.

* Add WhatsApp Number
* Add WhatsApp Default Message
* Floating Button
* More options to come etc.


== Installation ==

1. Just log into your WordPress admin dashboard
2. Visit Dashboard > Sidebar > Plugins > Add New
3. Search for "TP Chat Lite"
4. Install and Activate the "TP Chat Lite" plugin
5. Visit Dashboard > Sidebar > WhatsApp Chat
6. Add a valid WhatsApp number & default message (Mandatory)
7. Use & Enjoy!!!
